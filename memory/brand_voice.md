## Tone

Welcoming, straightforward, playful, trustworthy. Like a friendly board game cafe owner — knowledgeable but not pretentious.

## Terminology

- "Pre-loved" not "used" or "secondhand"
- "Every game deserves a second turn" — the tagline
- "Sellers" and "Buyers" — clear role language
- "Board games" not "tabletop games" (simpler for general audience)

## Writing Rules

- No exclamation marks in UI copy (they feel desperate)
- European date format: dd.MM.yyyy (31.08.2026)
- 24-hour time: 14:30 (not 2:30 PM)
- Latvian time separator is a dot: 14.30 (not 14:30)
- Prices always with euro symbol: €12.99
- Use sentence case for headings (not Title Case)

## Error Messages

- Be specific about what went wrong
- Suggest what the user can do next
- Never blame the user ("Invalid input" → "Please enter a valid email address")
- Keep technical details out of user-facing messages

## Empty States

- Always include a clear CTA
- Make them encouraging, not apologetic
- "No games found — be the first to list one" not "Sorry, no results"

## Nordic Minimalist Aesthetic

- Clean, spacious layouts with breathing room
- Let content speak — don't decorate
- Trust is built through clarity, not flashiness
- Color is intentional: frost blue for trust, aurora orange for action
